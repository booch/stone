Functions
=========


Function Call
-------------

You can call a function by name, using syntax that's pretty familiar.
There are a few built-in functions.

~~~ stone
identity(4)
#= Number.Integer(4)
~~~

~~~ stone
identity(TRUE)
#= Boolean(Boolean.TRUE)
~~~

~~~ stone
identity("abc")
#= Text("abc")
~~~

~~~ stone
min(1, 2)
#= Number.Integer(1)
~~~

~~~ stone
max(1, 2, 3)
#= Number.Integer(3)
~~~
